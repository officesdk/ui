# Officesdk Design

Welcome to Officesdk Design Component Library!

## Introduction

Officesdk SDK Design is a modern UI component library built with React and styled-components. It provides a series of high-quality, reusable components to help you quickly build beautiful user interfaces.

## Features

- **Modern Design** - Clean and beautiful design style
- **TypeScript Support** - Full type definitions
- **Flexible Styling** - Built with styled-components, easy to customize
- **Tree Shaking** - Supports on-demand loading
- **Ready to Use** - Complete default configuration
- **Well Documented** - Detailed component documentation and examples

## Tech Stack

- React 16.9+
- TypeScript
- styled-components
- Storybook

## Quick Start

### Installation

```bash
yarn add @officesdk/design
```

Or using npm:

```bash
npm install @officesdk/design
```

Note: This configuration is optional but recommended for optimal TypeScript IntelliSense support.

### Usage

```tsx
import { Button, initUIConfig } from '@officesdk/design';
import { lightTheme } from '@officesdk/design/theme';
import { iconRegistry } from '@officesdk/design/icons';

// Initialize configuration before React app starts (e.g., in index.tsx)
initUIConfig({
  theme: lightTheme,
  icons: iconRegistry,
});

function App() {
  return (
    <Button variant="solid" colorType="default">
      Click Me
    </Button>
  );
}
```

**Recommended**: Use `initUIConfig()` for simpler global configuration. It avoids Provider nesting and works even in non-React environments. For per-component-tree configuration, you can still use `UIConfigProvider`.
