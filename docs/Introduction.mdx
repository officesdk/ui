# Officesdk UI

Welcome to Officesdk UI Component Library!

## Introduction

Officesdk SDK UI is a modern UI component library built with React and styled-components. It provides a series of high-quality, reusable components to help you quickly build beautiful user interfaces.

## Features

- **Modern Design** - Clean and beautiful design style
- **TypeScript Support** - Full type definitions
- **Flexible Styling** - Built with styled-components, easy to customize
- **Tree Shaking** - Supports on-demand loading
- **Ready to Use** - Complete default configuration
- **Well Documented** - Detailed component documentation and examples

## Tech Stack

- React 16.9+
- TypeScript
- styled-components
- Storybook

## Quick Start

### Installation

```bash
yarn add @officesdk/ui
```

Or using npm:

```bash
npm install @officesdk/ui
```

### TypeScript Configuration

For better type resolution, add the following paths to your `tsconfig.json`:

```json
{
  "compilerOptions": {
    "baseUrl": ".",
    "paths": {
      "@officesdk/ui/theme": ["./node_modules/@officesdk/ui/dist/theme/index.d.ts"],
      "@officesdk/ui/utils": ["./node_modules/@officesdk/ui/dist/utils/index.d.ts"],
      "@officesdk/ui/icons": ["./node_modules/@officesdk/ui/dist/icons/index.d.ts"],
      "@officesdk/editor-sdk-core/shared": [
        "./node_modules/@officesdk/editor-sdk-core/types/shared.d.ts"
      ]
    }
  }
}
```

Note: This configuration is optional but recommended for optimal TypeScript IntelliSense support.

### Usage

```tsx
import { Button, UIConfigProvider, createUIConfig } from '@officesdk/ui';
import { lightTheme } from '@officesdk/ui/theme';
import { iconRegistry } from '@officesdk/ui/icons';

const config = createUIConfig({
  theme: lightTheme,
  icons: iconRegistry,
});

function App() {
  return (
    <UIConfigProvider config={config}>
      <Button variant="solid" colorType="default">
        Click Me
      </Button>
    </UIConfigProvider>
  );
}
```
